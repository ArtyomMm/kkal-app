# Welcome Screen hero assets

Place the 3 transparent PNG hero images here. Next.js serves this folder at
`/images/welcome/...`, and the Welcome Screen references them by these exact names:

- `steak.png`
- `beer.png`
- `pizza.png`

Recommended: square, transparent-background PNGs (~512–1024px).

Note: files under the project-root `./assets/` folder are NOT served by Next.js.
Static assets must live in `public/` to be reachable over HTTP.
